# Инструкция: как пользоваться расписанием встреч

**1. Сначала впиши своё имя** (поле «Вы:» в правом верхнем углу).
Это важно: имя автоматически подставляется как организатор новой встречи, а удалить встречу может только тот, чьё имя совпадает с полем «Организатор» — если не заполнить или ошибиться в написании, потом не сможешь удалить свою же встречу.

**2. Посмотреть расписание**
- Переключатель «День / Неделя» — вид на один день или на всю неделю.
- Стрелки `‹` `›` — листать вперёд/назад.

**3. Добавить встречу**
- Кнопка «+ Новая встреча».
- Заполни: тема, организатор (уже подставится твоё имя), участники через запятую, дата, время начала и конца.
- Нажми «Сохранить». Если кто-то из участников уже занят в это время — появится сообщение с указанием, кто и с кем занят; встречу нужно перенести.

**4. Посмотреть детали встречи**
Кликни по самой карточке встречи (в дне или в неделе) — откроется попап с темой, временем, организатором, участниками и датой создания. Закрыть — кнопкой «Закрыть» или кликом мимо попапа.

**5. Удалить встречу**
Кнопка «Удалить» видна на карточке в дневном виде. Сработает только если твоё имя в поле «Вы:» совпадает с организатором этой встречи.

**Важно помнить:** все встречи видят все, у кого есть ссылка на этот инструмент — это общий календарь команды, а не личный.
